Q4   What is the use of an iframe tag?
Ans: The iframe in HTML stands for inline frame.The iframe tag defines a rectangular 
     region within the document in which the browser can display a separate document,
     including scrollbars and borders.The iframe is basically used to show a webpage 
     inside the current web page.The 'src' is used to specify the URL of the document 
     that occupies the iframe.
