02     What is a Bootstrap Container, and how does it work?
Ans:- Containers are the most basic layout element in Bootstrap and are required when using our default grid system. Containers are used to contain, pad, and                  (sometimes) center the content within them. While containers can be nested, most layouts do not require a nested container

03     What are the default Bootstrap text settings?
Ans:- Bootstrap Default Settings Bootstrap uses a default font-size of 16px, and its line-height is 1.5. The default font-family is "Helvetica Neue", Helvetica, Arial, sans-         serif. In addition, all <p> elements have margin-top: 0 and margin-bottom: 1rem (16px by default).

04     What do you know about the Bootstrap Grid System?
Ans:- Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

05     What is the difference between Bootstrap 4 and Bootstrap 5.
Ans:- Bootstrap 5 includes custom form controls. Bootstrap 4's form controls used whatever presets were available in each browser. The form controls in Bootstrap 5 will          have a significantly more consistent appearance and feel across all browsers due to their custom design.

10     What is a bootstrap card and how would you create one?
Ans:- A card is a flexible and extensible content container. It includes options for headers and footers, a wide variety of content, contextual background colors, and          powerful display options.
         In its simplest form, a Bootstrap card is a .card class with the .card-body class nested inside. This creates a card with a rounded border inside which you can place          text, images, links, and other Bootstrap elements.
